# adblock_tester_-dns_filtering
Teste avançado de DNS Filtering e AdBlock (Pi-hole, AdGuard, NextDNS, uBlock, AdBlock Plus). Executa dezenas de checks em camadas (DNS, browser, CNAME, fingerprinting), gera pontuações, métricas em tempo real e relatórios para avaliar a eficácia do bloqueio de rastreadores e anúncios.
